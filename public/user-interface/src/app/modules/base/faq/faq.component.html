Why isn't the company I'm searching for appearing?

There are a few reasons why this may occur:

  • The company is not publicly traded, or not regulated by the SEC. 8-K forms are only filed by US-based securities that are publicly traded.
  • Their 8-K form has not been reviewed and added to the results.
  • They have not filed an 8-K form yet, or chose not to file one at all.
  • The 8-K form they filed during the SVB fallout was not related to SVB.

If you think you found a mistake, please send a contact request.

What is the search powered by?

The user interface is an Angular frontend and uses Algolia to power the search index.

You can find the source code for this user interface and all of the supporting code for this project on GitHub.
